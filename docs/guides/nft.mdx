# Create an NFT Community DAO

Follow the steps below to [create a new NFT Community DAO](#create-a-new-nft-community-dao) or [create an NFT Community DAO using an existing NFT contract](#create-an-nft-dao-using-an-existing-nft-contract). For more info on NFT Community DAOs, visit the [concepts page](../concepts/about.mdx#nft-community-daos).

## Prerequisites

Before you can create an NFT DAO, make sure you have completed the following:
- Download Google Chrome
- Install [the Terra Station Chrome Extension](https://docs.terra.money/learn/terra-station/Download/terra-station-extension)
- An NFT minter address

## Connect your wallet to Enterprise

1. Sign in to your Terra Station wallet using the [Terra Station Browser Extension](https://setup-station.terra.money/). 

2. To connect your wallet to Enterprise, click the wallet icon in the upper right corner of the screen. 

3. Select **Terra Station Wallet**. 

4. If needed, click **approve** in the Terra Station extension. 

You are now connected to your wallet. Before you continue, make sure your [network is set to the **mainnet**](https://docs.terra.money/learn/terra-station/networks).

## Create a new NFT Community DAO

1. Click on the plus sign on the left side of the screen and select **NFT Community DAO**. 

2. Input a name for your DAO and upload a logo. Click **Next**.

3. Enter your NFT name, NFT symbol, and provide an address for your NFT minter. Click **Next**.

4. Define your [threshold](../concepts/config.mdx#threshold), [quorum](../concepts/config.mdx#quorum), [unlocking period](../concepts/config.mdx#unlocking-period), and [vote duration parameters](../concepts/config.mdx#vote-duration). Click **Next**.

<Admonition type="caution" title="Updating parameters">

After creation, [general parameters](../concepts/config.mdx#general-parameters) can only be updated using a [DAO configuration proposal](../concepts/governance.mdx#dao-configuration-proposals). 

</Admonition>

5. Enter any social media links you wish to add. Click **Next**.

6. Input the password for your Terra Station browser extension wallet and click **Post**.

Congratulations! You've just made an NFT Community DAO!

## Create an NFT Community DAO using an existing NFT contract

Follow the steps below to make a DAO using an existing [CW-721 NFT contract address](https://github.com/CosmWasm/cw-nfts/blob/main/packages/cw721/README.md).

1. Click on the plus sign on the left side of the screen and select **NFT Community DAO**. 

2. Input a name for your DAO and upload a logo. Click **Next**.

3. Select **Yes, find my existing NFT** and enter the NFT contract address. Click **Next**.

4. Define your [threshold](../concepts/config.mdx#threshold), [quorum](../concepts/config.mdx#quorum), [unlocking period](../concepts/config.mdx#unlocking-period), and [vote duration parameters](../concepts/config.mdx#vote-duration). Click **Next**.


<Admonition type="caution" title="Updating parameters">

After creation, [general parameters](../concepts/config.mdx#general-parameters) can only be updated using a [DAO configuration proposal](../concepts/governance.mdx#dao-configuration-proposals). 

</Admonition>

5. Enter any social media links you wish to add. Click **Next**.

6. Input the password for your Terra Station browser extension wallet and click **Post**.

Congratulations! You've just made an NFT Community DAO!
